# WHOOP Dashboard — D3.js Implementation
DSA 301 Project 2 — Data Visualization

This repository contains the D3.js implementation of Dashboard 3 
(Weekly Load + Performance) from the WHOOP Health Dataset mobile app project.

## How to run
npm install d3-node d3
node whoop_dashboard3.js

## Output
The script generates whoop_dashboard3.svg which can be converted to PNG using cairosvg.
